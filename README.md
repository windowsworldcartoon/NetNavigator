# Installation
To install NetNavigator, follow these steps:
1. Download the latest version of NetNavigator from the [GitHub releases page](https://github.com/windowsworldcartoon/NetNavigator/releases)
2. Run the installer and follow the on-screen instructions.
3. Once the installation is complete, open the NetNavigator application.
